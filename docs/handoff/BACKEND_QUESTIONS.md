### Q1. (예시) pgvector 임베딩은 어떻게 생성하나요?

**작성자**: B팀
**작성일**: 2025-11-15
**우선순위**: High
**상태**: Answered

**질문 내용**:
업로드된 이미지의 `embedding` 컬럼 값을 어떻게 생성해야 하나요? CLIP 모델을 직접 실행해야 하는지, 외부 API를 사용하는지 궁금합니다.

**답변** (A팀):
임베딩 생성은 2단계로 나뉩니다:

1. **Phase 1 (현재)**: 임베딩 없이 업로드 허용
   - `embedding` 컬럼은 `NULL`로 두고 자산 저장
   - 유사도 검색은 나중에 구현

2. **Phase 2 (추후)**: 비동기 임베딩 생성
   - 업로드 완료 후 Celery 작업 큐에 임베딩 생성 작업 추가
   - Desktop GPU Worker에서 CLIP 모델 실행 (Ollama 또는 별도 서비스)
   - 생성된 임베딩을 PostgreSQL에 UPDATE

당장은 `embedding=None`으로 구현하시고, Phase 2에서 비동기 처리를 추가하면 됩니다.
