2. **솔루션**: Sparklio는 회의 영상 하나로 PPT·상세페이지·인스타 광고·쇼츠를 한 번에 만들어준다

### Slide 2: 문제 정의 - 마케팅 팀의 현실

**제목**: "마케팅 팀, 이런 고민 있으시죠?"

**문제 3가지**:
1. **회의는 넘쳐나는데 정리가 안 된다**
   - 주간 회의, 기획 회의, 클라이언트 미팅...
   - 회의록 작성, 핵심 메시지 추출, 타깃 정의가 전부 수동 작업

2. **채널별 산출물이 따로 논다**
   - PPT는 디자이너, 상세페이지는 카피라이터, 광고는 또 다른 팀
   - 같은 제품인데도 메시지·톤·컬러가 제각각

3. **브랜드 일관성이 깨진다**
   - 각 채널별로 따로 작업하다 보니 브랜드 가이드 놓치기 쉬움
   - 광고 규제, 과장 표현 검토도 수동

**비교 테이블**:
| 기존 방식 | Sparklio 방식 |
|---------|--------------|
| 회의 → 회의록 작성(수동) | 회의 URL → AI 자동 요약 |
| 브리프 작성(수동) | Meeting AI가 키메시지·타깃 추출 |
| PPT·상세·광고·쇼츠 개별 제작 | 멀티 에이전트가 한 번에 생성 |
| Figma/Miro에서 수동 정리 | Concept Board에서 자동 비교 |

**멘트**:
> "마케팅 팀은 매일 회의를 하지만, 회의 내용을 실제 캠페인으로 만드는 데까지 시간이 너무 오래 걸립니다.
> 게다가 PPT, 상세페이지, 인스타 광고, 쇼츠... 각 채널을 따로 만들다 보면 메시지도 제각각이 되죠.
> 우리는 이 문제를 해결하고 싶었습니다."

### Slide 3: 솔루션 - Sparklio AI Marketing Studio

**제목**: "Sparklio가 해결합니다"

**솔루션 핵심**:
```
회의 URL 입력
    ↓
Meeting AI가 자동 요약 (핵심 메시지, 타깃, 문제/해결 포인트)
    ↓
멀티 에이전트가 2-3개 콘셉트 생성
    ↓
각 콘셉트별로 4종 산출물 자동 생성
  - Presentation (3-5 슬라이드)
  - Product Detail Page
  - Instagram News Ad (카드 3종)
  - Shorts Script (20-30초)
    ↓
Concept Board에서 한 눈에 비교
```

**특징**:
- ✅ **한 번의 입력으로 모든 채널 커버**
- ✅ **여러 콘셉트를 동시 생성해서 비교 가능**
- ✅ **브랜드 일관성 자동 유지 + 광고 규제 자동 검토**

**멘트**:
> "Sparklio는 YouTube 회의 영상이나 Zoom 녹화 파일 하나만 주면,
> AI가 자동으로 요약하고, 2-3가지 마케팅 콘셉트를 뽑아냅니다.
> 그리고 각 콘셉트별로 프레젠테이션, 상세페이지, 인스타 광고, 쇼츠 스크립트를
> 한 번에 만들어서 '컨셉보드'라는 화면에서 비교할 수 있게 해줍니다."

---

### Slide 4: 핵심 플로우 개요

**제목**: "작동 원리: Meeting → Campaign → Multi-Output"

**상세 플로우**:

#### Step 1: Meeting From URL
- YouTube/Zoom 링크 입력
- yt-dlp로 자막 다운로드 (또는 Whisper STT로 오디오 → 텍스트)
- Meeting 레코드 생성

#### Step 2: Meeting AI 요약
- MeetingSummaryAgent 실행
- 출력:
  - 회의 제목
  - 한 줄 요약
  - 핵심 키메시지 3-5개
  - 타깃 페르소나
  - 주요 문제/해결 포인트

#### Step 3: Campaign Brief 생성
- StrategistAgent 실행
- 캠페인 목표, 타깃, 핵심 메시지, 톤앤매너 정의

#### Step 4: Concept 생성
- ConceptAgent가 2-3개 콘셉트 생성
- 예:
  - Concept A: "상큼한 하루 리프레시"
  - Concept B: "아이와 함께하는 비타민 간식"

#### Step 5: Multi-Output Generation
각 Concept별로:
- PresentationAgent → 3-5 슬라이드
- ProductDetailAgent → 상세페이지 텍스트
- InstagramNewsAdAgent → 카드 3종
- ShortsScriptAgent → 20-30초 스크립트

#### Step 6: Concept Board
- 모든 Concept와 산출물을 한 화면에서 비교
- 카드 클릭 → 해당 콘셉트의 산출물 상세 보기

**멘트**:
> "전체 플로우는 이렇습니다.
> 회의 URL을 입력하면, Meeting AI가 요약하고, Strategist Agent가 캠페인 브리프를 만들고,
> Concept Agent가 여러 콘셉트를 뽑아냅니다.
> 그러면 각 콘셉트별로 슬라이드, 상세페이지, 인스타 광고, 쇼츠 스크립트가 자동으로 만들어지고,
> 이걸 '컨셉보드'라는 화면에서 한 눈에 비교할 수 있습니다."

### Slide 5: 차별점 - 챗 기반 원페이지 스튜디오

**제목**: "모든 작업이 하나의 화면에서"

**원페이지 스튜디오 철학**:
- 라우트: `/studio/demo`
- **좌측 (20%)**: 브랜드 선택, Meeting 리스트
- **중앙 (50-60%)**: 결과물 표시 (Concept Board, Slides, Detail, Instagram, Shorts)
- **우측 (20-30%)**: Chat 패널 (Sparklio Assistant) ← **모든 액션의 시작점**

**Chat = 오케스트레이터**:
- 사용자는 자연어로 작업 지시
- Chat이 백엔드 파이프라인 호출
- 실시간 상태 Narration (예: "회의 요약 중...", "슬라이드 생성 중...")
- 다음 행동 버튼 제안 (예: `[컨셉보드 열기] [슬라이드 보기]`)

**멘트**:
> "Sparklio의 가장 큰 특징은 '챗 기반 원페이지 스튜디오'입니다.
> 모든 작업이 하나의 화면 안에서 이루어지고,
> 우측의 Chat이 모든 워크플로우를 오케스트레이션합니다.
> 사용자는 '이 회의로 캠페인 만들어줘'라고 말하면,
> Chat이 알아서 Meeting 분석하고, 콘셉트 만들고, 산출물 생성해서 보여줍니다."

**제목**: "기술 아키텍처"

- Whisper STT (OpenAI API / faster-whisper)

- **고성능 작업**: OpenAI GPT-4o, Google Gemini 2.0 Flash

  - CopywriterAgent, DesignerAgent

- ComfyUI (Stable Diffusion / Flux)
- ffmpeg 7.1.2 (영상 조립)

#### Frontend
- Next.js 14, React, TypeScript
- Polotno Editor (슬라이드 프리뷰)

#### Infrastructure
- Mac mini (100.123.51.5): Backend, DB, Redis, MinIO
- RTX Desktop (100.123.51.6): Whisper STT, Ollama, ComfyUI

## 4. 컨셉보드까지 - 발표용 핵심 문장 정리

### 문제 정의 (30초)
> "마케팅 팀은 매일 회의를 하지만, 회의 내용을 실제 캠페인으로 만드는 데까지 시간이 너무 오래 걸립니다. 게다가 PPT, 상세페이지, 인스타 광고, 쇼츠... 각 채널을 따로 만들다 보면 메시지도 제각각이 되고, 브랜드 가이드도 놓치기 쉽습니다."

### 솔루션 (30초)
> "Sparklio는 YouTube 회의 영상이나 Zoom 녹화 파일 하나만 주면, AI가 자동으로 요약하고, 2-3가지 마케팅 콘셉트를 뽑아냅니다. 그리고 각 콘셉트별로 프레젠테이션, 상세페이지, 인스타 광고, 쇼츠 스크립트를 한 번에 만들어서 '컨셉보드'라는 화면에서 비교할 수 있게 해줍니다."

### 차별점 (30초)
> "Sparklio의 가장 큰 특징은 '챗 기반 원페이지 스튜디오'입니다. 모든 작업이 하나의 화면 안에서 이루어지고, 우측의 Chat이 모든 워크플로우를 오케스트레이션합니다. 사용자는 '이 회의로 캠페인 만들어줘'라고 말하면, Chat이 알아서 Meeting 분석하고, 콘셉트 만들고, 산출물 생성해서 보여줍니다."

### 컨셉보드 (30초)
> "가장 핵심적인 화면이 바로 이 '컨셉보드'입니다. 여러 콘셉트를 카드 형태로 나란히 보여주고, 각 카드를 클릭하면 해당 콘셉트의 슬라이드, 상세페이지, 인스타 광고를 바로 볼 수 있습니다. 마케터가 A안과 B안을 비교하면서 방향을 결정할 수 있게 돕는 거죠."

### 데모 결과 (30초)
> "방금 보신 것처럼, 회의 URL 하나로 2분 30초 만에 2개 콘셉트, 8종의 마케팅 산출물이 만들어졌습니다. 모든 산출물은 광고 규제 검토를 통과했고, 브랜드 가이드도 자동으로 반영되었습니다. Sparklio는 마케팅 팀의 시간을 절약하고, 여러 콘셉트를 빠르게 비교할 수 있게 하며, 브랜드 일관성을 자동으로 유지해줍니다."
