## 1. 요구사항

### 1.1 데모 시나리오

- **주제**: 제주 감귤 젤리 신제품 런칭 기획 회의
- **내용**: 마케팅 기획 관련 대화
- **언어**: 한국어
- **길이**: 3-10분 권장 (너무 길면 처리 시간 증가)

### 1.2 좋은 테스트 영상 조건

- 한국어 음성이 명확함
- 배경 소음이 적음
- 비즈니스/마케팅 관련 내용
- 다운로드 가능 (비공개 아님)

## 5. 주의사항

### 5.1 저작권

- 데모용으로 사용 시 저작권 문제 확인 필요
- 가능하면 직접 제작한 영상 사용 권장
- 또는 CC 라이선스 영상 사용

### 5.2 가용성

- YouTube 영상은 언제든 삭제/비공개 전환 가능
- 발표 직전 URL 접근 가능 여부 반드시 확인
- 로컬 백업 파일 준비 권장

## 6. 추천: 직접 제작

가장 안전한 방법은 **직접 테스트용 영상 제작**:

### 6.1 간단한 제작 방법

1. Zoom/Google Meet으로 팀 회의 녹화
2. "제주 감귤 젤리 기획 회의" 시나리오 대화
3. 5분 이내로 핵심 내용만
4. YouTube에 비공개로 업로드
5. URL을 테스트용으로 사용

### 6.2 시나리오 예시

```
[회의 내용 예시]
- 제품명: 제주 감귤 젤리
- 타깃: 30-40대 엄마
- 핵심 메시지: 국내산 제주 감귤 100%, 합성 첨가물 무첨가
- 차별점: 어린이도 안심하고 먹는 비타민 간식
- 출시 시기: 2025년 상반기
```
