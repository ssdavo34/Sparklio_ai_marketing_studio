**문서 버전**: v1.0
**작성일**: 2025-11-26
**작성자**: A팀 (QA/문서)
**목적**: 발표자가 따라할 상세 데모 스크립트

## 1. 개요

### 1.1 데모 목표
> **"회의 한 번으로 캠페인 풀패키지"** - AI가 회의 녹음을 분석하여 4종 마케팅 에셋을 자동 생성

### 1.2 데모 구성
| 단계 | 내용 | 예상 시간 |
|------|------|----------|
| 인트로 | 서비스 소개 | 30초 |
| Step 1 | Meeting From URL | 60초 |
| Step 2 | Campaign 생성 | 90초 |
| Step 3 | Concept Board 탐색 | 90초 |
| Step 4 | Shorts 미리보기 | 60초 |
| 마무리 | 요약 및 Q&A | 30초 |
| **총** | | **6분** |

### 1.3 데모 환경
- **URL**: `http://localhost:3000/studio/demo`
- **Backend**: `http://100.123.51.5:8000`
- **브라우저**: Chrome (최신, 캐시 클리어)

>
> "Sparklio는 **회의 한 번**으로 이 모든 걸 **5분 안에** 만들어냅니다. 직접 보여드릴게요."

### 3.1 Step 1: Meeting From URL (0:30 - 1:30)

#### 화면
- Chat 입력창에 YouTube URL 입력

#### 발표자 멘트
> "먼저 회의 녹음이 필요합니다. 오늘은 제주 감귤 젤리 신제품 기획 회의 영상을 준비했어요."
>
> *(URL 붙여넣기)*
>
> "YouTube URL을 붙여넣고 엔터를 누르면..."
>
> *(전송)*
>
> "AI가 영상을 다운로드하고, 음성을 텍스트로 변환하고, 핵심 내용을 분석합니다."
>
> *(진행상황 표시되는 동안)*
>
> "지금 Whisper AI가 음성을 인식하고 있어요. 45분 회의 영상도 1-2분이면 분석됩니다."

#### 화면 동작
1. Chat 입력창 클릭
2. URL 붙여넣기: `https://youtu.be/[테스트URL]`
3. Enter 키 또는 전송 버튼
4. Chat 메시지 표시:
   - "회의 영상을 다운로드 중입니다..."
   - "음성을 텍스트로 변환 중입니다..."
   - "회의 내용을 분석 중입니다..."
   - "회의 요약이 완료되었습니다!"
5. 중앙 뷰: Meeting Summary 표시

#### 예상 시간
- 실제 처리: 30초
- 설명 포함: 60초

#### Fallback
- 처리 시간 길어질 경우:
  > "실시간으로 처리하다 보니 네트워크 상황에 따라 시간이 조금 걸릴 수 있어요. 실제 서비스에서는 캐싱으로 더 빠르게 처리됩니다."

### 3.2 Step 2: Campaign 생성 (1:30 - 3:00)

#### 화면
- Meeting Summary 표시 상태

#### 발표자 멘트
> "회의 요약이 완료됐습니다. 보시면 핵심 메시지들이 정리되어 있어요."
>
> *(Summary 내용 가리키며)*
>
> "'국내산 제주 감귤 100%', '합성 첨가물 무첨가', '어린이도 안심하고 먹는 비타민 간식'... 회의에서 나온 중요한 포인트들입니다."
>
> "이제 이 내용으로 마케팅 캠페인을 만들어볼게요."
>
> *(캠페인 만들기 버튼 클릭)*
>
> "버튼 하나로 AI가 알아서 캠페인을 기획합니다."

#### 화면 동작
1. Meeting Summary 내용 확인
2. [캠페인 만들기] 버튼 클릭
3. Chat 메시지 표시 (SSE 실시간):
   - "캠페인 브리프를 작성 중입니다..."
   - "콘셉트를 생성하고 있어요... Concept A: '상큼한 하루 리프레시'"
   - "Concept A 기반 발표자료 생성 중..."
   - "Concept A 기반 상세페이지 생성 중..."
   - "Concept A 기반 인스타 광고 생성 중..."
   - "Concept A 기반 쇼츠 스크립트 생성 중..."
   - "모든 산출물이 준비되었습니다!"
4. 중앙 뷰: Concept Board 자동 전환

#### 발표자 멘트 (진행 중)
> "지금 여러 AI 에이전트들이 동시에 일하고 있어요."
>
> "StrategistAgent가 캠페인 전략을 세우고, ConceptAgent가 마케팅 콘셉트를 만들고, CopywriterAgent가 광고 카피를 쓰고 있습니다."
>
> "각 전문 분야의 AI가 협업하는 거죠."

#### 예상 시간
- 실제 처리: 60-90초
- 설명 포함: 90초

#### Fallback
- LLM 응답 느릴 경우:
  > "AI가 고민 중이네요. 품질 좋은 결과를 위해 꼼꼼히 작업하고 있습니다."

### 3.3 Step 3: Concept Board 탐색 (3:00 - 4:30)

#### 화면
- Concept Board 표시 (2-3개 컨셉 카드)

#### 발표자 멘트
> "자, 캠페인이 완성됐습니다!"
>
> "보시면 2개의 마케팅 콘셉트가 생성됐어요."
>
> *(Concept A 가리키며)*
>
> "첫 번째는 '상큼한 하루 리프레시'. 제주 감귤의 상큼함을 강조하는 컨셉이에요. 톤 앤 매너는 밝고 경쾌한 느낌."
>
> *(Concept B 가리키며)*
>
> "두 번째는 '아이와 함께하는 비타민 간식'. 엄마 타깃, 안심을 강조하는 따뜻한 톤이에요."
>
> "각 콘셉트별로 4종의 마케팅 에셋이 준비되어 있습니다. 하나씩 볼까요?"

#### 화면 동작
1. Concept A 카드 마우스 오버 (하이라이트)
2. [슬라이드 보기] 클릭 → Slides Preview
3. 슬라이드 넘기기 (1→2→3)
   > "발표자료입니다. 표지, 문제제기, 솔루션, 기능소개, CTA까지 완전한 구성이에요."
4. [컨셉보드로 돌아가기] 클릭
5. [상세 보기] 클릭 → Detail Preview
   > "상세페이지입니다. 히어로 영역부터 가격표까지, 랜딩페이지 전체가 만들어졌어요."
6. [컨셉보드로 돌아가기] 클릭
7. [인스타 보기] 클릭 → Instagram Preview
   > "인스타그램 광고 3종입니다. 피드용, 스토리용, 캐러셀 광고까지."

#### 예상 시간
- 90초

## 4. Q&A 예상 질문

### 4.1 기술 관련

**Q: 어떤 AI 모델을 사용하나요?**
> "텍스트 생성은 Gemini 2.0 Flash를 사용하고 있습니다. 음성 인식은 OpenAI의 Whisper, 음성 합성은 Microsoft의 Edge TTS를 사용해요. 모두 최신 AI 기술입니다."

**Q: 처리 시간이 얼마나 걸리나요?**
> "회의 길이에 따라 다르지만, 보통 1-2분 안에 모든 에셋이 생성됩니다. 45분 회의 기준이에요."

**Q: 한국어만 지원하나요?**
> "현재는 한국어에 최적화되어 있지만, 다국어 지원도 계획 중입니다."

### 4.2 비즈니스 관련

**Q: 가격은 어떻게 되나요?**
> "데모 버전은 무료로 체험하실 수 있고, 정식 출시 시 요금제를 공개할 예정입니다."

**Q: 생성된 콘텐츠의 저작권은?**
> "생성된 모든 콘텐츠는 사용자에게 귀속됩니다. 상업적 이용도 자유롭습니다."

### 4.3 기능 관련

**Q: 콘셉트를 수정할 수 있나요?**
> "네, 생성된 콘셉트와 에셋 모두 편집 가능합니다. AI가 초안을 만들고, 사람이 다듬는 협업 방식이에요."

**Q: 이미지도 자동 생성되나요?**
> "네, AI 이미지 생성 기능도 준비 중입니다. 현재 데모에서는 이미지 프롬프트를 제공해드리고 있어요."

- `/backup/concept-board.png` 스크린샷 표시
