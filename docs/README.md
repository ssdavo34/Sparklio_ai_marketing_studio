# 문서 (Documentation)

이 폴더는 Sparklio AI Marketing Studio 프로젝트의 문서를 포함합니다.

## 문서 구조

- **API.md**: API 문서 및 엔드포인트 설명
- **Architecture.md**: 시스템 아키텍처 및 설계 문서
- **UserGuide.md**: 사용자 가이드
- **DeveloperGuide.md**: 개발자 가이드
- **Deployment.md**: 배포 가이드

## 문서 작성 가이드

문서는 마크다운 형식으로 작성하며, 다음 원칙을 따릅니다:

1. 명확하고 간결한 설명
2. 실용적인 예제 코드 포함
3. 스크린샷 및 다이어그램 활용
4. 정기적인 업데이트 및 검토
